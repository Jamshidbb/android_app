# Gradle Dependency Cache

This branch contains pre-downloaded Gradle dependencies.

## Quick Start

```bash
# Checkout this branch
git checkout gradle-dependency-cache

# Run the setup script
./.gradle-cache/setup-cache.sh

# Go back to your working branch
git checkout -

# Build your project
./gradlew build
```
